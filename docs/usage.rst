Usage
=====

Installation
------------

.. code-block:: bash

    pip install .    # install from local source directory
    pip install mikufetch

Running Mikufetch
-----------------

.. code-block:: bash

    mikufetch

Options
-------

--no-art   — disable the Miku ASCII art
--no-color — disable ANSI colors
--json     — output system info as JSON only
